#### Инструкции по установке
1. Скачайте архив;
2. Для запуска авто-тестов в Terminal Intellij IDEA открыть новую сессию и ввести команду:
   `gradlew clean test --info`
   где:
   `--info` - Развернутая информация о пройденных тестах;
